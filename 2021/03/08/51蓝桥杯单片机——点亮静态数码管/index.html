2.数码管

要点亮一个数码管，要知道一个位码和段码；
（位类似一个旅店的位置，段类似一个这个房间的钥匙）
位码：想点亮数码管的位置
端码：想点亮的内容
要想点亮一个数码管，要先确定位码（确定位置），然后在确定断码（确定要点亮的内容）

段码

共阳数码管段码
unsigned char code SEG_CODE[] = {0xc0,0xf9,0xa4,0xb0,0x99,0x92,0x82,0xf8,0x80,0x90};

位码

当二进制的1在什么位置时，什么位置的数码管亮。
